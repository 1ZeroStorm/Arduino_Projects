# version 4 with website and arduino IDE integration

## Dependencies
- install ```PubSubClient``` by Nick O'Leary in the arduino IDe

## How to run
1) run the .ino code via arduino IDE
- CHANGE THE WIFI AND PASS
- Select the correct COM port (Windows) or /dev/ttyUSB0 (Linux/Mac)
- Set baud rate to 115200
- Click "Connect"
- adjust the power supply roughly about 5.30-5.40 volts
2) access the web from the serial monitor
